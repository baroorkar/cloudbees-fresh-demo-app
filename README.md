# CloudBees Fresh Demo Application

This repository contains a sample Node.js application used for the CloudBees CI demo assignment.

## Pipeline workflow

1. Checkout code from GitHub
2. Provision a Kubernetes build agent
3. Validate the Node.js application
4. Build a Docker image
5. Push the image to Docker Hub
6. Deploy the application to Amazon EKS
7. Validate the deployment

## Technology stack

- CloudBees CI
- Amazon EKS
- Kubernetes
- Jenkins Pipeline
- Docker
- Node.js
- GitHub
